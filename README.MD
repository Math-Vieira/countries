<h2>:bookmark_tabs: Description</h2>
<p>A simple project created to practice react concepts.</p>

<h2> :video_game: How to start</h2>

Clone this repository.

```bash
git clone https://github.com/Math-Vieira/countries
```
Install the dependencies.

```bash
npm install
```
Start the server.

```bash
npm start
```
